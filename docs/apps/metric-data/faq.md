# Metrics Data FAQ

<div class="app-empty-state app-empty-metric">

## Does Metrics Data support multiple AdSense accounts?

Yes. Account switching is supported.

## Which OAuth flow is implemented?

Authorization Code with PKCE.

## Where are tokens stored?

In Apple Keychain.

## Which date filters are available?

7D, 30D, 90D, This Month, Last Month.

## Is there widget support?

Yes. Widget snapshots are shared through app group storage.

## Does it use subscriptions or one-time purchase?

Project docs currently contain both one-time purchase and subscription language in different documents. Confirm current product policy before publishing storefront text.
