# Neon Vision Editor Changelog

## v1.0.0

Initial public multi-platform launch.

- shipped on macOS, iPadOS, and iOS
- introduced native editor core with syntax highlighting
- added optional AI assistance flow
- published beta channel via [TestFlight](https://testflight.apple.com/join/YWB2fGAP)
- published stable release via [App Store](https://apps.apple.com/de/app/neon-vision-editor/id6758950965)

## v1.0.1

Stability and quality pass.

- improved editor responsiveness in larger files
- refined UI consistency across iPhone and iPad layouts
- improved cross-device feel and visual parity

## v1.1.0 (planned)

- deeper editor workflow refinements
- additional polish for AI assist interaction
- broader feature documentation updates and examples
