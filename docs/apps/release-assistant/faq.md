# Release Assistant FAQ

<div class="app-empty-state app-empty-release">

## Does Release Assistant replace release scripts?

No. It orchestrates and safeguards script-based release pipelines.

## Is dry-run mode available?

Yes. Dry-run validation is a first-class workflow mode.

## What date format is required?

`YYYY-MM-DD`.

## Can I use it with GitHub Enterprise?

Yes. Enterprise host-aware auth and workflow modes are supported.

## Does it include audit output?

Yes. Every run includes a structured audit header in output logs.
