# Vistral Installation

## Requirements

- Xcode with multi-platform SDK support
- Apple platform runtime for the target you test (iOS/macOS/tvOS/watchOS/visionOS)

## Local Setup

1. Generate/open project from the existing project configuration.
2. Select platform-specific scheme (for example `Vistral-iOS` or `Vistral-macOS`).
3. Build and run on simulator/device.
4. Import sample export files to validate dashboard flow.
