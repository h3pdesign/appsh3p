## Application Purpose

Vistral is a privacy-first personal data visualization app. Its purpose is to import user-provided data exports and turn them into explainable insights and dashboards while keeping processing primarily on-device.

Primary user outcomes:

- import structured exports (CSV/XML/JSON)
- view trends and comparisons in clear visual dashboards
- inspect explainable correlations with confidence context
- keep sensitive personal analytics workflows local-first
