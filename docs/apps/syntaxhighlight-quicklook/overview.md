## Application Purpose

SyntaxHighlightQuickLook is a macOS Quick Look extension for previewing text-based files with syntax highlighting. Its purpose is to provide readable, color-coded previews directly in Finder without opening a full editor.

Primary user outcomes:

- preview code/text files quickly in Finder
- keep syntax colors for better scanability
- support common development-oriented file types in a lightweight extension
