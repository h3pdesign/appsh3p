# X-Newsbook Installation

## Requirements

- iOS 17+ / iPadOS 17+ / macOS 14+
- network access for feed updates

## Install Steps

1. Install TestFlight or internal distribution build.
2. Launch X-Newsbook and allow required permissions.
3. Add at least one source feed in onboarding.

## Initial Setup

- choose preferred text size
- enable offline cache for saved stories
- pick sync mode for cross-device reading
