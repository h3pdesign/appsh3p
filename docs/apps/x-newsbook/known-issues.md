# X-Newsbook Known Issues

## Feed Source Rate Limits

Some feeds may refresh slower under provider rate limits.

### Workaround

- reduce refresh interval
- prioritize key sources

## Offline Cache Growth

Large saved queues can increase local storage usage.

### Workaround

- prune older saved stories periodically
- cap offline retention in settings
