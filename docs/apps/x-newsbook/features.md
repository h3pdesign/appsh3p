# X-Newsbook Features

## Reading Experience

- distraction-minimized reader mode
- adjustable typography and spacing
- theme-aware layout for light/dark usage

## Content Management

- curated source lists
- saved stories and quick return queue
- unread tracking and lightweight tagging

## Sync and Continuity

- resume reading across Apple devices
- saved state sync for queue and bookmarks
