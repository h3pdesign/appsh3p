# Support

Use this section for support channels, issue reporting, legal/security references, and troubleshooting entry points for all h3p apps.

## Quick Start

1. Check app-specific docs first:
   - [All Apps](/apps/index)
   - Overview, Installation, Known Issues, and FAQ pages per app
2. If the issue remains, open a support request:
   - [Support and Feedback](/support/support-and-feedback)
3. For security issues, use private disclosure:
   - [Security and Trust](/policies/security-and-trust)

## Support Channels

- Support and issue guidance: [Support and Feedback](/support/support-and-feedback)
- Documentation issues: [appsh3p issues](https://github.com/h3pdesign/appsh3p/issues)
- Security policy and contact: [Security and Trust](/policies/security-and-trust)
- Privacy and terms:
